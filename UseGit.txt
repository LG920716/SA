再來記住以下步驟：
1.新建資料夾當作一個 repository
2.右鍵叫出 git bush
3.輸入 git init 使該資料夾變成 repository

再來是重要指令：
1.git status：查看目前狀態
2.git add <檔案名稱>：將此檔案加入git的追蹤項目 (git add . 就是加入所有檔案)
3.git commit：依目前狀態存檔 (簡單一點就用 git commit -m "<輸入敘述>")
4.git log：查看每次的commit
5.git remote add <自訂名稱> <網址>：加入一個remote
6.git remote：查看當前有的remote
7.git push -u <remote名稱> <branch名稱>：把當前的branch推送到該remote上 (-u會讓之後預設都為這個remote)
8.git clone <網址>：複製當前網址的檔案
